open-data

This repository contains material I have used for teaching courses on the law and licensing of open data. The contents should be self-explanatory, consisting (at the moment) of:

  • A half-day segment in the ODI's Open Data in Practice course:
    • slides (in PDF format)
  • A half-day course on Open Data Law and Licensing:
    • slides (in powerpoint format) to which I talk
    • a handout (with a PDF version) containing more detailed notes

If you have any problems with these documents please raise as a git issue if possible, or send comments to me at

The repository also contains source markdown files and a makefile that runs pandoc to generate the final versions. At present the makefile is likely to work only on my personal laptop.
